Detailed Description

template<size_t SIZE = DEFAULT_RING_CAPACITY>
class porth::PorthRingBuffer< SIZE >

A high-performance SPSC Lock-Free Queue.

Optimized for Zero-Copy data transfer between Hardware and Application. This implementation enforces strict cache-line separation between the producer (head) and consumer (tail) to maximize throughput on many-core systems.

Template Parameters
SIZENumber of descriptors. Must be a power of two to allow the use of bitwise AND instead of the integer division (modulo) instruction.
