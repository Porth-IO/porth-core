Detailed Description

template<typename T>
class porth::PorthRegister< T >

The atomic "brick" of the Porth-IO Hardware Abstraction Layer.

  • Enforces 64-byte cache alignment to prevent false sharing and utilizes Acquire/Release semantics for jitter-free hardware synchronization.
  • Template Parameters
    TThe integral type of the register (must be trivially copyable).
