Detailed Description

template<typename T>
class porth::PorthRegister< T >

The atomic "brick" of the Porth-IO Hardware Abstraction Layer.

Enforces 64-byte cache alignment to prevent "False Sharing," where adjacent hardware registers would otherwise compete for the same L1 cache line. It utilizes Acquire/Release semantics to provide a jitter-free synchronization bridge between the Sovereign Logic Layer and physical hardware.

Template Parameters
TThe integral type of the register. Must be trivially copyable to ensure raw bit-level compatibility with MMIO space.
