Detailed Description

template<size_t Capacity = 1024>
class porth::PorthShuttle< Capacity >

The Zero-Copy Orchestrator for high-performance DMA.

Handles the "Placement New" logic to ensure the RingBuffer lives inside pinned HugePage memory. This eliminates memory copy overhead between the hardware logic layer and user-space applications.

Template Parameters
CapacityThe number of descriptors in the ring. Defaults to 1024.

  • This constructor allocates a 2MB HugePage and maps the PorthRingBuffer structure directly onto that memory region. Because HugePages are 2MB aligned, the ring buffer is guaranteed to be 64-byte aligned.
