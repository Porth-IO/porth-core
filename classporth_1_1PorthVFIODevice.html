Detailed Description

Manages physical PCIe hardware via the Linux VFIO interface.

This class implements the "Sovereign Takeover" of a PCIe device. It claims the device's IOMMU group, enables bus mastering, and maps the hardware registers (BARs) directly into the logic layer's address space.
