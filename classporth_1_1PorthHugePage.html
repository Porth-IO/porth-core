Detailed Description

PorthHugePage: Strictly Linux RAII wrapper for HugePage memory. Optimized for InP/GaN DMA transfers.

Parameters
sizeThe requested size (will be rounded up to 2MB boundary).

MAP_PRIVATE: Not shared with other processes (unless forked). MAP_ANONYMOUS: Not backed by any file. MAP_HUGETLB: Use the 2MB HugePage pool. MAP_LOCKED: Pin pages in RAM; prevent the kernel from swapping them.
