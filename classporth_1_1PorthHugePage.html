Detailed Description

RAII wrapper for pinning memory to 2MB HugePage boundaries with NUMA affinity.

Optimized for InP/GaN DMA transfers, this class ensures memory is physically co-located on the same NUMA node as the hardware device to eliminate cross-socket interconnect latency.
