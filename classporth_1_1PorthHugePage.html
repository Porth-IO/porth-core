Detailed Description

Strictly Linux RAII wrapper for HugePage memory.

Optimized for InP/GaN DMA transfers, this class provides pinned memory that is resistant to swapping and TLB overhead. It implements a robust fallback mechanism for environments without pre-allocated HugePage pools.

inlineexplicit

Constructor: Allocates pinned memory with HugePage support.

  • Attempts to allocate memory from the HugePage pool first. If the kernel is not configured for HugePages, it falls back to standard pages while retaining the MAP_LOCKED attribute to prevent jitter-inducing swaps.
  • Parameters
    sizeThe requested size in bytes.
    Exceptions
    std::runtime_errorIf both allocation attempts fail.

Attempt 1: The "Fast Path" MAP_PRIVATE: Copy-on-write mapping. MAP_ANONYMOUS: Not backed by any file (RAM only). MAP_HUGETLB: Utilize the 2MB HugePage pool. MAP_LOCKED: Pin pages to RAM to prevent kernel swapping.
