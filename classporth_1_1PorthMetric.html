Detailed Description

Statistical analysis engine for ultra-low latency timing.

  • Supports standard metrics, jitter analysis, and automated markdown reporting. This class uses a pre-allocated buffer to ensure recording samples does not trigger heap allocations, preserving the determinism of the test run.
