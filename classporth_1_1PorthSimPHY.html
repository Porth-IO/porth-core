Detailed Description

Emulates PCIe physical layer effects for compound semiconductor interconnects.

  • This class implements the Task 4.3 Thermal/Power Feedback Loop, providing a high-fidelity simulation of propagation delays, jitter, and signal degradation caused by physics-level fluctuations in InP/GaN hardware.
